# 光线追踪渲染器
基于 Taichi 语言实现的光线追踪渲染器，支持漫反射、镜面反射、硬阴影和棋盘格地面。

## 项目简介
本次实验实现了光线追踪渲染器，包含以下核心功能：
光线与物体求交：支持球体（漫反射/镜面材质）和无限大平面（棋盘格纹理）
光照计算：环境光 + 漫反射光照，带硬阴影检测
镜面反射：支持递归/迭代式光线追踪，模拟镜面反射效果
交互控制：实时调节光源位置和光线弹射次数

通过本次实验，深入理解了光线追踪的核心算法流程，包括光线生成、场景求交、光照计算和递归追踪。
