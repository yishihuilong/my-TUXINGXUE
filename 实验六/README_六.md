# 可微渲染
基于 Taichi 实现的可微分渲染器，通过梯度下降自动优化三维场景中的光源位置，使其与目标光照效果匹配。

## 项目简介
本次实验实现了基于物理的可微分渲染管道，包含以下核心功能：

正向渲染：基于 Lambertian 漫反射模型渲染球体
目标图像生成：使用预设光源位置生成参考图像（Ground Truth）
可微分管道：利用 Taichi 的自动微分机制计算 Loss 对光源位置的梯度
梯度优化：使用 Adam 优化算法迭代更新光源三维坐标，最小化 MSE Loss

通过本次实验，深入理解了可微分渲染的核心原理，包括渲染管道的可微性设计、自动微分的应用，以及基于梯度的逆向渲染优化方法。
