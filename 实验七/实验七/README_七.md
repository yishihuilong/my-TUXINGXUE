# 质点弹簧模型
基于 Taichi 语言实现的质点弹簧系统模拟器，支持三种积分方法（显式欧拉、半隐式欧拉、隐式欧拉）的实时对比演示。

## 项目简介
本次实验实现了基于质点弹簧系统的布料模拟，包含以下核心功能：
物理建模：质点 + 结构弹簧模型，模拟布料的弹性形变
三种积分方法：显式欧拉（极易发散）、半隐式欧拉（稳定）、隐式欧拉（带阻尼）
交互控制：实时切换积分方法、暂停/恢复模拟、重置布料
可视化：3D 粒子与弹簧线框渲染，支持鼠标拖拽视角

通过本次实验，深入理解了不同数值积分方法对物理模拟稳定性的影响，以及 GPU 并行计算的编程范式。
